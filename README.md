# goggibiography.github.io

Инструкции по замене логотипа на 3D модель

- Положите вашу 3D модель в папку `assets` и назовите файл `model.glb` (формат GLB/gltf предпочтителен).
- В корне `index.html` подключён компонент `model-viewer` — он загрузит `assets/model.glb` и покажет его вместо растрового логотипа NERV.
- Поддерживаемые режимы: управление камерой (drag), авто-вращение и AR (если устройство поддерживает).

Если хотите временно вернуть растровый логотип, просто удалите или закомментируйте элемент `<model-viewer>` в `index.html`.

Если модель не загружается: убедитесь, что путь `assets/model.glb` корректен и что браузер поддерживает загрузку локальных ресурсов (локально через файловую систему некоторые браузеры блокируют CORS — в таком случае используйте локальный сервер, например `npx http-server` или `python -m http.server`).
